# Returns & Exchanges Policy

## How to return
Start a return from the Orders page within 30 days of delivery. A prepaid return
label is provided for faulty items; for change-of-mind returns the customer pays
return postage.

## Condition
Items must be unused, in original packaging, with tags attached. Returns that
arrive damaged due to poor packaging may be partially refunded.

## Exchanges
Exchanges for a different size or colour are free where stock is available.
If the replacement is out of stock we issue a refund instead.
